
### Advanced Integration
```python
# Example: Loading prompts programmatically
import json

def load_prompt(filename):
    with open(f'prompts/{filename}', 'r', encoding='utf-8') as file:
        return file.read()

expert_prompt = load_prompt('expert-mode-framework.md')
```
